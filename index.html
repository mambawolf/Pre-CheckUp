What is Pre-CheckUp

As the name is self-explainatory, Pre-CheckUp is a web application that is used by doctors and medical practioners for entering and storing patients information which he answers during the preliminary medical examination queries. This application is thus an alternative to the traditional priscript based medical records and replaces it to store and also manage data records for future reference. A must use application for doctors.
